# Evcard Analytics Next

Next.js + shadcn/ui + Lucia + DrizzleORM + Vercel Postgres + OracleDB

## Description

Evcard Analytics Next.js Server

## Installation

Install Node.js 20.x.x LTS

```bash
$ cp .env.example .env.local
```

Setup .env.local file

```bash
$ npm install
```

## Running the app

```bash
npm run build

npm run start
```
